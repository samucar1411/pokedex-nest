# Run on development 

### 1. Clone the repository
### 2. Run this command
```
yarn install
```
### 3. Must have Nest CLI

```
npm i -g @nest/cli
```
### 4. Set the database
```
docker-compose up -d
```
### 5. Rebuild database with seed
```
http://localhost:3000/api/v2/seed
```

## Stack used

* MongoDB
* Nest
